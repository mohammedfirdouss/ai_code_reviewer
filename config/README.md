# Configuration Files

This directory contains additional Wrangler configuration files for different deployment scenarios.

## Files

- **`wrangler.workers.toml`** - Configuration for Workers deployment
- **`wrangler.pages.toml`** - Configuration for Pages deployment

## Usage

Reference these configs when deploying:

```bash
# Deploy with specific config
wrangler deploy --config config/wrangler.workers.toml
```
